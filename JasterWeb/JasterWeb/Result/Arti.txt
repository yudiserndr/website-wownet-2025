Responsive
-----------------------------
Responsive adalah salah satu perkembangan dari dunia Website yang fungsinya menjadikan Website
bila diakses di HP / Layar kecil tetap terstruktur seperti website pada ukuran desktop (laptop / komputer)
Artinya. Jika Web anda sudah responsive. Web anda tidak berantakan saat di akses di HP.


SearchGoogle
------------------------
Ini adalah hasil Optimasi SEO dari Jaster. yang menaikkan Website anda sesuai dengan Keyword yang anda
Target.
Sedangkan digambar seo-result.jpg itu adalah hasil bahwa web anda sudah terindex dengan lancar di google
*Jika tidak ada file ini di folder result itu berarti masih dalam tahap pengindexan. Kami akan kasih kabar jika sudah terindex

Mobile Friendly
-----------------------------
Ini adalah sertifikat langsung dari Google yang artinya Website anda jika diakses via HP / Layar kecil
tetap rapi dan clickable. artinya semua tombol / gambar / object yang bisa di klik sesuai dengan ukuran 
jari normal manusia pada umumnya. Hampir sama seperti Responsive tetapi Mobile Friendly lebih
ke isi Konten.
